## Primeira instalação

**Dependências:** É preciso ter algumas dependências instaladas antes de continuar

- Node.js e NPM;
- Yarn.

&nbsp;

## Etapas
### 0. Instale as dependências do projeto

```
yarn install
```

### 1. Para subir a aplicação, basta rodar

```
yarn start
```
